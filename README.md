Attendance Alarm is an AI-powered productivity app designed for students. The app listens for important lecture keywords such as **attendance**, **quiz**, **test**, or **submission** and instantly alerts the user in real time.

Built using Python and Vosk speech recognition, the application provides a lightweight offline solution for students who want to stay alert during long lectures and avoid missing critical announcements.

## Features

* Real-time speech keyword detection
* Multiple custom trigger words
* Stealth headphone-only alarm mode
* Popup notifications
* Attendance/event logging
* Dark and light themes
* Custom alarm sounds
* Modern desktop GUI
* Offline speech recognition (no internet required)

## Technologies Used
